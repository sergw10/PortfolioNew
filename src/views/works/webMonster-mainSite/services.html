Поддержка сайтов в Днепропетровске и по всей Украине


Получение консультаций

В период эксплуатации сайта у его владельца регулярно возникают вопросы по его обслуживанию. Для регулярных консультаций мы за каждым владельцем сайта закрепляем личного менеджера с которым можно связываться по телефону, E-mail или через SKYPE


Обновление содержания веб-страниц

Владелец сайта имеет возможность самостоятельного обновления содержания сайта с помощью панели управления. Если по какой то причине есть желание, чтобы эти обновления выполнял Разработчик сайта, то мы можем организовать регулярное обслуживание по редактированию веб-страниц.


Создание и хранение резервных копий сайта

На сайте ежедневно автоматически создаются резервные копии сайта. В качестве дополнительной услуги мы можем скачать эту копию на свой сайт и хранить ее для страховки. Чем больше резервных копий будет храниться, тем больше вероятности сохранить сайт.
